Conway's Game of Life Sandbox

What can it do?

The application is a sandbox which allows the user to draw on a grid which follows rules of the Conway's Game of Life Algorithm. What kind of unique patterns can be created from the simple rules followed by each spot on the grid?

Why make this?

I like the idea of emergence, complex behaviors coming from simple principles. It's a concept I'd like to explore more seriously in future projects, in the form of an application or a game design project.

Conway's Game of Life uses only the following 3 rules:

  1. Filled cells in the grid with more than 2 or 3 neighbors persist.
  2. Empty cells in the grid with 3 live neighbors become filled.
  3. All other filled cells become or stay empty.

How is it made?

The application was written using the python language, specifially Python 3.8.3. Outside of the latest python version, the pygame module had to be installed before it could be written. To avoid having to run it through a console window, I used PyInstaller to wrap it into an executable file.
